My SPA index.html

Demonstrates spa-static-server use and configuration

This page should appear for all routes, allowing angular's HTML5 mode ui-router to activate the appropriate state, i.e.:

  • / (home)
  • /foo
  • /foo/bar

Static asset files are served before the SPA "catch-all" route. If a file does not exist on the filesystem, a 404 will be returned.

  • /styles.css
  • /file-not-found.txt (404 status)

The server can also provide a "SPA Boot Config" JSONP callback for run-time configuration:

  • /spa-boot.js?callback=main
